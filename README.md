# הייטק בית שמש - פלטפורמת Full Stack כישרונות טכנולוגיים

אתר תדמית Full Stack להצגת כישרונות הטכנולוגיים בבית שמש ומשיכת חברות הייטק לגיוס מפתחים מקומיים.

## תכונות עיקריות

### 🏠 דף הבית
- Hero section גדול עם כותרת חזקה
- הצהרת משימה קצרה וויזואליות של חדשנות וקהילה
- קריאה לפעולה (CTA) המובילה לדף "צור קשר"
- סטטיסטיקות אנימטיביות של מפתחים, חברות ותוכניות
- סיפורי הצלחה מקומיים

### 👥 דף כישרונות
- הצגת מספר המפתחים והטכנולוגיות הזמינות
- בוטקמפים ותוכניות חינוך
- סטארטאפים וחברות מקומיות
- סיפורי הצלחה עם מדדים
- תרשימים ואינפוגרפיקה ידידותיים למובייל

### 🔐 מערכת ניהול (פרטי)
- התחברות מאובטחת
- ניהול מאמרים (הוספה/עריכה/מחיקה)
- ניהול תוכן האתר
- עדכון סטטיסטיקות מפתחים
- ניהול פניות מחברות
- צפייה באנליטיקה (תעבורה, טפסי טפסים)

### 📰 דף חדשות ומאמרים
- רשימת מאמרים על סצנת ההייטק בבית שמש
- אפשרויות חיפוש וסינון
- תבנית מאמר SEO-friendly
- עיצוב רספונסיבי מלא לגלישה במובייל

### 📞 דף צור קשר
- טופס פנייה לחברות טכנולוגיות
- שדות: שם חברה, איש קשר, סיבת פנייה, טכנולוגיות נדרשות
- אישור אוטומטי בדוא"ל
- CTA קבוע "צור קשר" בכל הדפים

## 🚀 התקנה והרצה

### דרישות התקנה

#### שרת
```bash
# התקנת תלותי
npm install

# הפעלת השרת בפיתוח פיתוח
npm run dev

# הפעלת במצב ייצור
npm start
```

#### התקנה מקומית
1. התקן את MongoDB מקומית או באמצעות MongoDB Atlas
2. הגדיר את משתני הסביבה בקובץ `.env`
3. הגדיר את חשבון הדוא"ל בהגדרות הסביבה
4. התקן את Node.js ו-npm

#### גישה למערכת ניהול
1. גש ל-`admin/login.html`
2. שם משתמש: `admin`
3. סיסמה: `beitshemeshtech2023`

## 🔧 כלים וספריות

### Frontend
- **Font Awesome** - אייקונים
- **Chart.js** - גרפים במערכת ניהול
- **Vanilla JavaScript** - ללא תלות בספריות חיצוניות
- **CSS Grid & Flexbox** - עיצוב רספונסיבי
- **Intersection Observer API** - אנימציות על גלילה

### Backend
- **Express.js** - פריימוורק ווב לאפליקציות RESTful
- **MongoDB** - מסד נתונים NoSQL עם Mongoose ODM
- **JWT** - אימות משתמשים והרשאות מאובטחת
- **bcryptjs** - הצפנת סיסמאות באופן חד-צופן
- **Nodemailer** - שליחת אימיילים אוטומטית
- **Helmet.js** - אבטחת אבטחה עבור Express
- **Morgan** - לוגר בקשות HTTP עם Morgan
- **CORS** - תמיכה Cross-Origin Resource Sharing
- **Compression** - דחיסת תגובות HTTP
- **express-rate-limit** - הגנלת קצב פניות
- **dotenv** - ניהול משתני סביבה
- **multer** - העלאת קבצים בטפסים
- **express-validator** - ולידציה וסינון קלט
